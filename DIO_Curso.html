Introdução

(Voltar)

HTML (HyperText Markup Language) - é uma linguagem de marcação utilizada na construção de páginas na web.

O código do HTML é o esqueleto do site.

CSS (Cascading Style Sheets) - é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc) a um documento web.

CSS é a parte visual do site. Essa parte do arquivo pode ser interna ou externa.


Tag's

(Voltar)

Em HTML, tag's são os elementos de marcação que definem a estrutura na página web.

As tag's são escritas com sinais de menor que e maior que (< >).

(< p >) - representa um parágrafo.

(< head >) - contém informações sobre o documento, que não serão exibidas no navegador para os usuários.

(< title >) - permite esepecificar o título do documentos, que será exibidio na aba da janela do navegador.

(< body >) - contém o corpo do documento, que é exibido no navegador

(< ! - - comentário - ->) - essa tag é utilizada para adicionar comentários no código, que não serão exibidos no navegador.

(< hr >) - define uma mudança temática no conteúdo.

(< br >) - faz com que pule uma linha entre as escritas no documento, forçando a quebra de linhas dentro do parágrafo ou entre parágrafos.

(< a >) - essa tag define um hiperlink, usado para vincular de uma página para outra.

(href=" ") - é o atributo mais importante da tag "< a >", pois ele que carrega o destino do link.

(< sub >) - deixa o texto ou a palavra subscrito (ex.: A2).

(< sup >) - deixa o texto ou a palavra sobrescrito (ex.: A2).

(< blockquote >) - é utilizado para deixar o texto em "destaque" quando contém uma citação que seja de referente a outro documento, local, autor/site.


Estilização Textual

(Voltar)

Essas tag's alteram a aparência e/ou o significado do texto.

Para criar títulos e subtítulos existem as tag's heading que vão de 1 a 6;

(< h1; h2; h3; h4; h5; h6 >) - elas seguem a ordem de importância, sendo h1 o título principal, e h6 o título de menor importância.

(< strong >) ou (< b >) - é utilizado para deixar o texto em negrito.

(< em >) ou (< i >) - deixa o texto em itálico.

(< small >) - irá diminuir o tamanho do texto.

(< mark >) - é utilizado para destacar o texto como um marca texto.

(< u >) -deixa o texto sublinhado.


Listas

(Voltar)

Existem dois tipos de tag's para criar listas.

(< ol >) - cria listas ordenadas, o que significa que será organizada em ordem numérica crescente.

Podemos mudar o modo da ordenação da lista, por exemplo em ordem alfabética (< ol type="a" >) ou com algarismos romanos (< ol type="I" >).

(< ul >) - essa ao contrário da anterior, cria listas sem ordenação, marcada apenas por bullets.

(< li >) - para adicionar os itens na lista essa tag deve ser acionada.
